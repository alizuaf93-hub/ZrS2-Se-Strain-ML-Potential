# Strain-Modulated Structural Stability in Se-Substituted ZrS2

This repository contains the data and scripts used in the study:

"Strain-Modulated Structural Stability and Dopant-Induced Lattice Distortion in Se-Substituted ZrS2: A Machine-Learning Potential Study"

## Authors
Zohaib Ali  
Khuram Ali

## Overview
This work investigates the structural stability and local lattice distortion induced by selenium substitution in ZrS2 under applied strain. Simulations were performed using the MatterSim machine-learning interatomic potential.

## Simulation Workflow

1. Structure relaxation
2. Supercell construction
3. Strain application
4. Selenium substitution
5. Structural relaxation
6. Energy–strain analysis
7. Bond length distribution analysis
8. Local distortion analysis

## Interatomic Potential
Simulations were performed using the pretrained MatterSim machine-learning interatomic potential (MatterSim-v1.0.0-1M).

## Data
The repository includes:

- relaxed structures
- strain-energy datasets
- bond-length distributions
- figure generation scripts
